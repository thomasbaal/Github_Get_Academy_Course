Javascript const

  • Variables defined with const cannot be Redeclared
  • Variables defined with const cannot be Reassigned.
  • Variables defined with const have Block SCope.

const PI = 3.141592653589793;
PI = 3.14; // This will give an error
PI = PI + 10; // This will also give an error

Javascript const must be assigned a value when they are declared

Example:

Correct:
const PI = 3.14159
Incorrect:
const PI;
PI = 3.14159

When to use const ?

As a general rule , always declare a variable with const unless you know that the value ill change.
Use const when you declare:
  • A new Array
  • A new Object
  • A new Function
  • A new RegExp

So what is an Array?


In Javascript, array is a single variable that is used to store different elements. Is is often used when we want to store list of elements and access them by a single variable. Unlike most languages where array is a reference to the multiple variable, in Javascript array is a single variable that stores multiple elements.

Redeclaring an existing var or let variable const, in the same scope, is not alowed:

Example:

var x = 2; - allowed const x = 2; - Not allowed, alreaddy declared. { let x = 2; - Allowed const x = 2; - Not allowed }

Reassigning an existing const variable , in the same scope, is not allowed:

Example:

const x = 2; - Allowed x = 2; - Not allowed var x = 2; - Not allowed let x = 2; - Not allowed const x = 2; - Not allowed { const x = 2; - allowed x = 2; - Not allowed var x = 2; - Not allowed let x = 2; - Not allowed const x = 2; - Not allowed }

Redclaring a variable with const, in another scope, or in another block, in allowed:

Example:

const x = 2; - Allowed { const x = 3; - Allowed } { const x = 4; - Allowed }
