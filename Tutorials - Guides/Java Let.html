Variables defined with let cannot be Redeclared

var variables can be reeclared.
let and const variable provides Block Scope in Javascript Variables declared inside a {} block cannot be accessed from outside the block:

Example

>

let x = 10;
let x = 2; // dette vil skape syntax error, already declared. console.log(x); console.log(x);

A var variable can be Redeclared enytime in the document, but might cause problems.
