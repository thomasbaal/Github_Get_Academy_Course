The Differences Between "innerHTML", InnerText and textContent.

element.innerText or node.innerText = Returns the text content of an element or node:

"element.innerText = text" or "node.innerText = text" = Sets the text content of an element or node.

Differences:

The innerHTML property returns: The text content of the element, including all spacing and inner HTML tags.

----

The innerText property returns:

just the text content of the element and all its children, without css hidden text spacing and tags, except script and style elements.

The textContent property returns: The text content of the element and all descendaces, with spacing and CSS hidden text, but without tags.
